AI Fitness Coach – Real-Time Posture Analysis
Overview

This project is a real-time computer vision system that tracks human posture and evaluates exercise form using pose estimation and biomechanical analysis.

Real-time pose detection using MediaPipe
Joint angle calculation using vector mathematics
Squat detection and repetition counting
Real-time posture feedback
Scalable design for multiple exercises
🛠 Tech Stack
Python
OpenCV
MediaPipe
NumPy
